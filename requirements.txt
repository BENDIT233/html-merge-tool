# 项目依赖包列表
# 以下是项目运行所需的Python依赖包及其版本

# Pillow: 用于图像处理，支持打开、操作和保存多种图像格式
# 版本锁定为9.2.0以确保兼容性
Pillow==9.2.0

# pythonnet: 提供Python和.NET之间的互操作性

# 版本锁定为3.0.5以确保与项目使用的.NET版本兼容
pythonnet==3.0.5

# eel: 用于创建桌面应用程序的Python库
# 提供基于Web技术的GUI界面功能
# 版本锁定为0.16.0以确保稳定性

eel==0.16.0

# 其他依赖说明:
